### 1.在文件开始时写块注释：

```
/** 
 * 作用于哪些页面
 * 注明代码业务/功能说明
 * @authors Your Name (you@example.org)
 * @date    2016-05-16
 * @version $Id$
 */ 

```

### 2.单个函数注释

```
/**
 * 函数功能简述
 *
 * 具体描述一些细节
 *
 * @param    {string}  address     地址
 * @param    {array}   com         商品数组
 * @param    {string}  pay_status  支付方式
 * @returns  void
 */

```

### 3.css注释的写法

```
/* Footer */

内容区

/* End Footer */

```
